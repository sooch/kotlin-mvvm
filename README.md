# Android MVVM Architecture with Kotlin

This is sample project.

## Environment

- Android Stuido 3.0 or higher
- Kotlin 1.0 or higher

This app perform the network access (GitHub API v3)

## Libraries

- Databinding Library
- Architecture Components
- RxJava2
- RxAndroid
- Dagger2
- Retrofit2
- Moshi
